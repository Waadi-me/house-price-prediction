Project Overview

What is this project?

A complete end-to-end machine learning pipeline built from scratch to understand and predict residential home sale prices.

This project takes raw King County housing data and transforms it into accurate price predictions using classical regression techniques. Every step — from exploratory analysis to model evaluation — is done transparently so you can understand exactly how the predictions are made.

The dataset contains 4,600 real home sales across cities like Seattle, Bellevue, Kirkland, and Renton. Features include physical attributes, location, condition, renovation history, and more.

Methodology

How it works

A reproducible 4-stage pipeline from raw CSV to prediction.

01
Data

Load & Explore

Import the CSV, examine distributions, plot correlations, and identify the features most predictive of sale price.

02
Engineering

Feature Engineering

Parse dates, compute house age, encode cities, flag renovations, and create ratio features to give models richer signals.

03
Training

Train & Evaluate

Split 80/20, train Linear and Polynomial models, measure RMSE, MAE, and R² on the held-out test set.

04
Predict

Predict & Interpret

Feed new house features into the best model, output price estimate, and inspect which features drove the prediction.
